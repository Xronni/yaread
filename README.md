YaRead — AI-Powered PDF Reader with AI Support



1\. Installation



Download the archive and extract it completely into a separate folder.

Important: do not move the yaread.exe file to the desktop on its own. It must remain in the same folder as the rest of the application files, otherwise the program will close with an error. If you want to create a desktop shortcut, right-click yaread.exe, select "Send to", and choose "Desktop (create shortcut)".



2\. Setup and AI Configuration



Launch the program and configure your preferred AI engine on the top panel:



Online Mode (quick start and low-spec PCs): Choose your provider from the list (OpenAI, Gemini, DeepSeek, or OpenRouter) and paste your API key. Note: The application uses your official API credits.



Offline Mode (local and private): Choose local .GGUF. Download the recommended qwen2.5-7b-instruct-q4\_k\_m.gguf model from Hugging Face. Place the file inside the "models" folder, which must be located in the application's root directory. After configuration, click "Open PDF" (or simply drag and drop your book into the window) and click "Start".



3\. Controls and Hotkeys



Spacebar — Play/Pause (starts/stops automatic text scrolling and background music).



Ctrl + F — Open quick text search bar.



Escape — Close search bar / clear active selections.



Alt + Mouse Highlight — Hold the Alt key and highlight any text fragment to call the "Ask AI" button for instant encyclopedic information.



Arrow keys (Up/Down) and PageUp/PageDown — Manual text navigation at any time.



4\. Soundtracks



The application dynamically plays built-in background tracks based on 12 different narrative emotions. The tracks are embedded in the application and cannot be changed.



Here is the list of tracks (emotions):

1. Narrative or Storytelling
2. Tension
3. Action or Climax
4. Resolution
5. Romance
6. Sadness
7. Joy
8. Mystery
9. Chase
10. Horror
11. Memories
12. Comedy

YaRead — программа для чтения PDF-файлов с поддержкой ИИ



1\. Установка



Загрузите архив и полностью распакуйте его в отдельную папку.

Важно: не перемещайте файл yaread.exe на рабочий стол отдельно. Он должен оставаться в той же папке, что и data, иначе программа завершит работу с ошибкой. Если вы хотите создать ярлык на рабочем столе, щелкните правой кнопкой мыши по yaread.exe, выберите «Отправить в», затем «Рабочий стол» (создать ярлык).



2\. Использование и настройка ИИ



Запустите программу и настройте предпочитаемый движок ИИ на верхней панели:



Онлайн-режим (быстрый старт и ПК с низкими характеристиками): выберите своего поставщика из списка (OpenAI, Gemini, DeepSeek или OpenRouter) и вставьте свой API-ключ. Примечание: приложение использует ваши официальные API-кредиты.



Автономный режим (локальный и частный): выберите локальный .GGUF. Загрузите рекомендуемую модель qwen2.5-7b-instruct-q4\_k\_m.gguf с Hugging Face. Поместите файл в папку "models", которая должна находиться в корневой директории программы. После настройки нажмите «Открыть PDF» (или просто перетащите книгу в окно) и нажмите «Старт».



3\. Управление и горячие клавиши



Пробел — Воспроизведение/Пауза (запускает/останавливает автоматическую прокрутку текста и фоновую музыку).



Ctrl + F — Открытие строки быстрого поиска текста.



Escape — Закрытие строки поиска/очистка активных выделений.



Alt + Выделение мышью — Удерживайте клавишу Alt и выделите любой фрагмент текста, чтобы вызвать кнопку «Спросить ИИ» для мгновенного получения энциклопедической информации.



Стрелки (Вверх/Вниз) и PageUp/PageDown — Ручная навигация по тексту в любое время.



4\. Саундтреки



Приложение динамически запускает встроенные фоновые треки на основе 12 различных повествовательных эмоций. Треки вшиты в приложение, и их нельзя изменить.

4. Развязка

6. Грусть
7. Радость
8. Тайна
9. Погоня

11. Воспоминания
